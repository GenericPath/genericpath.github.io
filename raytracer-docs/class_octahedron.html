Detailed Description

Class for Octahedron objects.

This class provides an Object which is an Octahedron centred at the origin with "radius" 1. That is, an Octahedron with vertices at \(\pm1\) along each axis, as illustrated below:

Octahedron-Ray intersection computation.

Todo:
The Cube-Ray intersection needs to be implemented as part of the assignment.

The intersection of a Ray with an Octahedron can be approached in a few ways. The basic constraint for a point on the surface of the base Octahedron is \(|x| + |y| + |z| = 1\). A simple, but not very elegant, way to approach this is to consider the eight faces independently, and intersect the ray with each of them in turn. For example, the face with negative \(Z\)- and positive \(X\)- and \(Y\)-values is defined by \( x + y - z = 1\).
